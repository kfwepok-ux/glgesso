# GL Gesso

Site estatico da GL Gesso preparado para deploy na Vercel.

## Deploy na Vercel

- Framework Preset: `Other`
- Build Command: vazio
- Output Directory: `.`

## Arquivos privados

O arquivo `database_structure.sql` e um backup MySQL do antigo WordPress e deve permanecer fora do GitHub e da Vercel. Ele esta listado no `.gitignore`.
